Задание №1

Почему код даёт именно такие результаты?

Дан код:

  • var a = 1, b = 1, c, d;
  • c = ++a; alert(c); // 2, к "а" присваиваем 1, дальше к "c" присваиваем ++а, два плюса перед "а", говрит о том, что нужно прибаваить 1 к "а" перед следующей операцией, "a" уже содержит 1, прибавляем еще 1 и получаем 2, затем выводим на экран "c"
  • d = b++; alert(d); // 1, к "b" присваиваем 1, дальше к "d" присваиваем b++, два плюса после "b", говрит о том, что нужно прибаваить 1 к "b" после следующей операции, т.е. "b" уже содержит 1, дальше "b", т.е. 1 присваевается "d", а затем только "b" инкроментируется, т.е. увеличивается на 1, выводим на экран "d"
  • c = (2+ ++a); alert(c); // 5, на даной строчки кода "a" = 2, ++ перед "а" говорит, о том, что нужно увеличить "а" на 1 до следующий операции, т.е. "а" будет содержать 3, 3+2=5, вывод на экран "с"
  • d = (2+ b++); alert(d); // 4, на даной строчки кода "b" = 2, т.к. выше по коду есть постфиксная форма записи b++(2). 2+2=4, выводим "d"
  • alert(a); // 3, Первоначально "а"=1, затем идет префиксная форма инкремента (++а) и "а" уже содержит 2, затем снова префиксная форма записи, где "а" уже содержит 3.
  • alert(b); // 3, Первоначально "b"=1, затем идет постфиксная форма инкремента (b++) и "b" уже содержит 2, затем снова постфиксная форма записи, где "b" уже содержит 3.
